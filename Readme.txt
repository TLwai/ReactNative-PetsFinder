 UI - use bottomtab, drawer navigation
 Database - restful web API, sqlite
 Notificaton - Firebase, restful web API,
 Map - google Map
 Camera - react-native-vision-camera
 The desing is quite simple: 
 1. Provide a UI to input the missing pets.
 2. Provide a UI to report the pets found.
 3. Provide a map to location the missing and found pets.
 4. Provide a setting page.
 5. Send & receive notification likes the pets found to the owner.


 Library Installation:
 npm install @react-navigation/bottom-tabs
 npm install @react-native-async-storage/async-storage
 npm install --save react-native-sqlite-storage
 npm install --save react-native-vector-icons
